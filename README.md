> WebStorm Theme for VS Code

## Overview

This extension integrates multiple WebStorm themes, but also creates a new WebStorm theme.

- WebStorm Darker
- WebStorm Darcula
- WebStorm Dark
- WebStorm New Dark
- WebStorm New Darcula
